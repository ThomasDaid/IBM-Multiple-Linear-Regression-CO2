# Multiple Linear Regression – CO2 Emissions Prediction

This project was developed as part of the IBM Data Science Professional Certificate.

The objective is to build a Multiple Linear Regression model to predict vehicle CO₂ emissions using engine characteristics.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Fuel Consumption dataset.
